# Gestión de Mantenimiento - SOLPACK (CMMS)

Sistema integral de gestión de mantenimiento preventivo y correctivo para plantas industriales.

## Tecnologías
- **Frontend**: React + Vite + Tailwind CSS
- **Backend**: Node.js + Express
- **Base de Datos**: PostgreSQL (Sequelize)
- **Despliegue Recomendado**: Vercel (Frontend), Render (Backend), Neon.tech (DB)

## Estructura
- `/client`: Aplicación frontend en React.
- `/server`: API backend en Node.js.

## Configuración Local
1. Instalar dependencias en ambas carpetas: `npm install`.
2. Configurar el archivo `.env` en `/server` con las credenciales de la base de datos.
3. Ejecutar `npm run dev` en ambas carpetas.
